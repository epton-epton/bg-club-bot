# Дизайн Mini App

## Референсы

- Карточки, отступы, типографика — мобильный health-app (скрин в чате)
- Палитра и настроение — логотип **Pair-a-dice Board game** (synthwave: синий + неон)

## Бренд

| Токен | Значение | Назначение |
|-------|----------|------------|
| `--brand-bg` | `#0a1028` | фон |
| `--brand-surface` | `#141d3a` | карточки |
| `--brand-cyan` | `#3de8ff` | основной акцент |

## Компоненты

- Карточки: radius 24px, мягкая тень + лёгкое neon-glow на акцентах
- Нижняя навигация: плавающая «пилюля», активная вкладка — cyan glow
- Шапка: логотип + приветствие

## Не в scope сейчас

- Тёмная/светлая тема переключателем (пока один тёмный бренд-стиль)
- 4-я вкладка «Столы» (MVP 2)
