## BGG — синхронизация каталога

**Когда:** вместе с админ-UI (MVP 5 или отдельный шаг раньше).

**Что:** кнопка «Синхронизировать с BGG» для админа.

**Поведение:**
- только игры с `bgg_id`
- повторный запрос к BGG API
- обновить: `bgg_rating`, `bgg_rank`, обложку, описание, игроков, время (по необходимости)

**Технически:** логика обновления уже есть в `import_game_from_bgg` при повторном импорте. Нужно:
- `POST /api/v1/games/bgg/sync` — все игры с BGG
- опционально `POST /api/v1/games/bgg/{id}/sync` — одна игра
- кнопка в админке Mini App
